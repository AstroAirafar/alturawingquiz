# AlturaWing Quiz — Ratan Tata

An educational web project featuring a short biography and an interactive quiz about Ratan Tata, including a case study on the Tata Nano.

## Features

* Informational landing page with references and videos
* 10-question multiple-choice quiz
* Instant results and answer review
* Simple admin dashboard for viewing quiz statistics

## Tech Stack

* Node.js + Express
* SQLite database
* HTML, CSS, JavaScript

## Run Locally

1. Clone the repository

   ```
   git clone https://github.com/AstroAirafar/alturawingquiz.git
   cd alturawingquiz
   ```

2. Install dependencies

   ```
   npm install
   ```

3. Create a `.env` file in the project root and add:

   ```
   PORT=3000
   ADMIN_PASSWORD=your_password_here
   SESSION_SECRET=your_random_secret_here
   ```

4. Start the server

   ```
   npm start
   ```

Open in browser:

```
http://localhost:3000
```
